🏋️‍♂️ Fitness | Premium Landing Page
A high-performance, responsive gym landing page built with React, Tailwind CSS v4, and Framer Motion. This project features a modern Gold and Charcoal aesthetic, dynamic content filtering, and advanced UI animations.

✨ Key Features
Dual-Theme Support: Seamless toggle between Light and Dark modes with persistent state management.

Dynamic About Carousel: A vertical, auto-sliding mission statement section powered by Framer Motion.

Service Discovery: A real-time search bar that filters premium gym services (Personal Training, Yoga, etc.).

Responsive Grid Layouts: Fully optimized for mobile, tablet, and desktop viewing.

Interactive Contact Form: Validated input fields with animated success feedback banners.

Modern Branding: Consistent use of bold typography (Orbitron) and high-contrast fitness palettes.

🛠️ Tech Stack
Frontend: React 19 + Vite 8

Styling: Tailwind CSS v4 (Alpha/Vite Integration)

Animations: Framer Motion

Icons: FontAwesome 6.4 (CDN)

Deployment: Vercel / GitHub Actions

📥 Installation & Setup

1. Clone the repository:

   git clone https://github.com/your-username/gym-landing-page.git

2. Install dependencies:

   npm install --legacy-peer-deps

3. Run in development mode:

   npm run dev
